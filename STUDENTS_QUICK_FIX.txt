================================================================================
                        TROUBLESHOOTING
================================================================================

BLANK PAGE?
  1. Check backend is running: npm run dev
  2. Check frontend is running: npm run dev
  3. Open browser console (F12) for errors
  4. Check Network tab for API response

STUDENTS NOT SHOWING?
  1. Make sure students exist in database
  2. Check if students are active (not deactivated)
  3. Try creating a test student account
  4. Check backend logs for errors

ERROR MESSAGE?
  1. Click [Retry] button
  2. Check backend console for details
  3. Verify database connection
  4. Restart backend

================================================================================
                        HOW TO DEBUG
================================================================================

1. OPEN BROWSER CONSOLE
   Press F12 → Console tab

2. LOOK FOR MESSAGES
   - "Students response: ..." - Shows API response
   - Error messages in red

3. CHECK NETWORK TAB
   - Look for /api/teacher/students request
   - Check response status (should be 200)
   - Check response body for data

4. CHECK BACKEND CONSOLE
   - Look for GET /api/teacher/students
   - Check for any error messages
   - Verify database queries
