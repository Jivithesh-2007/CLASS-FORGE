   - Create API key
   - Copy key (starts with sk-ant-)

   - Open: CLASS-FORGE/backend/.env

   - Replace with your actual key

   - Stop current backend (Ctrl+C)
   - Run: npm run dev

================================================================================
                            TROUBLESHOOTING
================================================================================

PROBLEM: "Failed to generate AI insights"
SOLUTION: 
  1. Restart backend: npm run dev
  2. Try again
  3. Check backend console for errors

PROBLEM: Backend won't start
SOLUTION:
  1. Check port 5001 is not in use
  2. Verify MongoDB is running
  3. Check .env file has all variables

PROBLEM: Can't login
SOLUTION:
  1. Make sure backend is running
  2. Check http://localhost:5001/api/health
  3. Verify MongoDB connection

PROBLEM: Claude API not working
SOLUTION:
  1. Check API key starts with sk-ant-
  2. Verify key is in .env file
  3. Restart backend after adding key
  4. Check you have available credits
