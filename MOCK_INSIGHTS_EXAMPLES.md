## Key Features of Mock Insights

✅ **Domain-Aware** - Analysis tailored to the idea's domain
✅ **Realistic Scores** - Scores vary between 75-95 for variety
✅ **Consistent Format** - Same structure for all ideas
✅ **Actionable** - Provides real recommendations
✅ **Professional** - Suitable for academic review
✅ **Instant** - No API delays
✅ **Free** - No costs

## Upgrading to Claude API

If you want even more personalized analysis:

1. Install: `npm install @anthropic-ai/sdk`
2. Get API key from: https://console.anthropic.com
3. Add to `.env`: `ANTHROPIC_API_KEY=sk-ant-your-key`
4. Restart backend

Claude API will provide:
- More specific analysis of YOUR idea
- Personalized recommendations
- Deeper insights into feasibility
- Better market analysis
- More detailed risk assessment

## How to Use

1. **Click eye icon** on any pending idea
2. **Wait for modal** to load (instant with mock)
3. **Review insights** - Read the analysis
4. **Add feedback** - Optional teacher feedback
5. **Approve or Reject** - Make your decision
